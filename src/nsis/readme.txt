The project is dependent by by NSIS install system
For download and more details visit http://nsis.sourceforge.net/Main_Page

The installer script is built over NSIS v2.46. To be able to create AREasy
install package you have to install NSIS on your workstation and than to
configure "build/profile<or.user.name>.properties" properties file, and to
set "deploy.nsis.path" property key with the installation path of NSIS
